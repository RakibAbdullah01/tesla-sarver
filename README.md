# tesla-server-side

# Project Description

This is a car selling website. To make this site I used HTML5, CSS3, Bootstrap5, ReactJS, MongoDb, express js, Firebase, Heroku.

# Functionality

- User Login , Registraion
- Dashboard for both Admin and User
- Animation
- Private Routing
- Login Redirect
- Manage Products
- Manage Orders
- Ratings and Review
